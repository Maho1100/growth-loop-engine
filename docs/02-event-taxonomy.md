## イベントカテゴリ

### ページ / 画面

| イベント名 | 説明 | 主要プロパティ |
|-----------|------|---------------|
| <!-- page.viewed --> | <!-- ページ表示 --> | <!-- url, referrer, title --> |

| イベント名 | 説明 | 主要プロパティ |
|-----------|------|---------------|
| <!-- button.clicked --> | <!-- ボタンクリック --> | <!-- button_id, label --> |

### システムイベント

| イベント名 | 説明 | 主要プロパティ |
|-----------|------|---------------|
| <!-- error.occurred --> | <!-- エラー発生 --> | <!-- error_code, message --> |

## 共通プロパティ

すべてのイベントに付与される共通フィールド。

| プロパティ名 | 型 | 説明 |
|-------------|---|------|
| event_id | string (UUID) | イベント一意識別子 |
| event_name | string | イベント名 |
| timestamp | string (ISO 8601) | 発生日時 |
| user_id | string | ユーザー識別子 |
| session_id | string | セッション識別子 |
| properties | object | イベント固有のプロパティ |
